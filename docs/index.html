Sports Reporter Tools

A small toolkit and research archive for AI-supported sports-journalism teaching. The aim is not to make AI write sports journalism for students, but to help writers see how their writing works on a reader.

The tools (SR1–SR4) are copiable prompts: take one, paste it into an AI assistant, and give it a match report, preview or feature to analyse. The supporting documents are organised in four streams — SR tools, SRD design notes, SRE evidence, and SRP papers — keeping practical tools, design rationale, test evidence and publishable drafts separate while preserving the development sequence.

New here? Start with Match Report Analyser (SR1), or the beginner route Match Report Coach (SR2). New in this version: the Feature Analysis Tool (SR4).

#Benchmark

The project includes a reproducible-testing protocol — a scoring rubric, case and run-log templates, and a model-comparison matrix — for validating tool runs across models. It is a working scaffold for contributors and lives in the repository rather than as library pages.
