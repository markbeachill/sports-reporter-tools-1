# Cases

Store benchmark case files here.

Use one folder per case when possible, for example:

- `MR-001-strong-football-report/`
- `MP-001-weak-match-preview/`
- `SR2-001-beginner-report/`
- `FT-001-sports-feature-profile/`

Each case folder should include a case note based on `../case-template.md` and, where permitted, the article text or excerpt being tested.
